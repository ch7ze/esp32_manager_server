UART Connection Info

UART (Universal Asynchronous Receiver-Transmitter) allows communication with ESP32 devices over serial connection.

  • Connect multiple ESP32 devices to a single serial port
  • Each device must send messages with a "device_id" field in JSON format
  • Example: {"device_id": "esp32-001", "temperature": "25.5"}
  • Messages are automatically routed to the correct device
