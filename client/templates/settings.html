ℹ️ UART Connection Info

UART (Universal Asynchronous Receiver-Transmitter) allows communication with microcontroller devices over serial connection.

  • Connect multiple microcontroller devices to a single serial port
  • Each device must send messages with a "device_id" field in JSON format
  • Example: {"device_id": "device-001", "temperature": "25.5"}
  • Messages are automatically routed to the correct device
